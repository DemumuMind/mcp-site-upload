# Post-release Hypercare

## Owner
- DevOps Lead

## Start / Deadline
- Start: 2026-03-16
- Deadline: ongoing

## Operating loop
1. Day 0-30: daily incident + KPI review.
2. Day 31-60: 3x weekly review.
3. Day 61-90: weekly review and stabilization report.

## Triage rules
- P0/P1 incidents get immediate owner assignment.
- User feedback is triaged within 24h.
- Weekly action review closes or re-prioritizes open items.
