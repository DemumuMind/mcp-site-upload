## Runbook first, then release

The first launch should always include a clear runbook with rollback thresholds and owner contacts.

A checklist does not slow delivery, it protects delivery velocity over time.

## Minimum launch checklist

If one item is missing, the release is not ready.

- Health metrics and alerts configured
- Error triage path documented
- Fallback behavior validated in staging
- On-call handoff completed
